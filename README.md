# 技术栈

- React
- React-Router
- Redux
- Vite
- TypeScript
- Ant Design
- Ant Design Charts
- Axios
- ESLint
- react-hook-form
- zod

# 运行

```shell
pnpm i
pnpm  run start
```

# 功能一览

- [√] 类似 `Next.js` 自动生成路由 、动态侧边栏菜单项、动态面包屑
- [√] 将 Websocket 轮训数据封装进 Redux
- [√] Dashboard 图表
- [√] 自定义 hook ，useObserver 监听页面元素变化
- [√] Item RESTful Api 增删改查、分页、当前页搜索、全局搜索
- [√] Item 表单 `react-hook-form` 结合 `zod` 方便验证
- [√] Axios 封装方便调用
- [√] 封装 Storage ，方便记忆登陆
- [√] 自定义日志，方便调试
- [√] Vite 优化打包体积，引入 Cdn，自定义插件 DNS 预解析
